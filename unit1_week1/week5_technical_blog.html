Ruby Classes


Ruby is a object-oriented Program. Everything on Ruby is an object. Ruby Class allows you to create Objects and use the Class Methods to perform actions. Below I have described how classes and Objects work together and how it can be used.


Let's have some fun with Classes and Objects. Let's say we have Data of TV Channels and the shows per channel and the what day/time it plays. If we want to output the TV Channel and the Shows and their timings. This can be achieved using Ruby Classes & Objects

class Tvshows
def initialize(shows)
@shows = shows
end
def showlist
@shows.each do |x,y|
puts "#{x} plays at #{y}"
end
return
end
end

In the above piece of code the @shows is called a 'instance variable'. Instance variables are available to all the methods for the intance of an object.As you see in the above example I haved used @shows in the showlist method to display the showlistings

Now, if we want to display the TV Channels and Show listings with timings we will create a Object called cbs. I will pass a hash as argument with Tv shows and their timings

cbs = Tvshows.new({"CSI" => "Mon 9:00 PM PST", "NCIS" => "Tues 8:00 PM PST"})

Now we will call the method showlist as

cbs.showlist


This will output:

NCIS plays at Tues 8:00 PM PST
CSI plays at Mon 9:00 PM PST
