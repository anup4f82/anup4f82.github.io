Ruby Methods


One of the cool things about Ruby is the Enumerable Module. Collection objects in Ruby reside in the Enumberable Module. Most of the searching and querying functionality used in Ryby comes from Enumerable. Some of the commonly used Methods provided by Enumerable are: collect,map,include?,sort,reduce,group_by


The map method is one of the most powerful enumerable operations in Ruby.map will always return an array as its output.Below I have described the Enumerable map method in detail and some of its uses:


Map

The Enumerable map method is used on a array to transform its receiver based on the rules set in the code block. The output of Enumerable map method is always an Array.

Syntax:

array.map{|item|block}

Example:

If we have an array , say

numbers = [1,2,3,4]

If we want to get an array that squares each number in the original array then we can use the map method:

new_numbers = [1,2,3,4].map{|x| x*2}

Output: new_numbers = [2,4,6,8]

Another example is, Lets say we have an array

names = ["tim","tom","sam","mike"]

If we want to capitalize every element of the array, we can use the map method as below:

new_names = ["tim","tom","sam","mike"].map{|x| x.capitalize}

Output: new_names = ["Tim","Tom","Sam","Mike"]

In the above cases we used the map method. Ruby also provides the map! method which is the destructive version of map. It will transform the array in place and return it.

e.g: In the above example where we capitalize every word of the array using the map method, the value of the original array 'names' will remain ["tim","tom","sam","mike"]

But if we use map! method, it will transform the original array as well:

names = ["tim","tom","sam","mike"]

new_names = names.map!{|x| x.capitalize}

Output:

names = ["Tim","Tom","Sam","Mike"]

new_names = ["Tim","Tom","Sam","Mike"]

As you can see the Enumerable map method is very powerful and can be a usefool method in programming!.
