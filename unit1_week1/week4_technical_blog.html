Ruby Methods


Ruby has a ton of methods you can use in your program to do complicated programming faster. Ruby methods lets you achieve taks which would usually take number of lines of code into a single line of code!


Below I have listed on how to use the Array Map method:


Map

The Array map method is used on a array to transform an Array based on the rules set in the code block. The output of Array map method is a transformed array.

Syntax:

array.map{|item|block}

Example:

If we have an array , say a = [1,2,3,4]. If we want to get an array that squares each number in the original array then we can use the map method:

[1,2,3,4].map{|x| x*2}

Output: [2,4,6,8]

Another example is, Lets say we have an array s = ["tim","tom","sam","mike"] and we want to capitalize every element of the array, we can use the map method as below:

["tim","tom","sam","mike"].map{|x| x.capitalize}
